# Источники

## Основные источники

- Базовая документация climate: https://esphome.io/components/climate/
- Документация Midea: https://esphome.io/components/climate/midea/
- Документация thermostat: https://esphome.io/components/climate/thermostat/

## Что из этого используется в проекте

Для текущего рабочего конфига напрямую используются:

- общий раздел `climate`
- специализированный раздел `midea`

Раздел `thermostat` нужен для понимания архитектуры climate в ESPHome, но текущий конфиг на нём не построен.

## Практический вывод

Если задача — повторить этот проект один в один, основной ориентир такой:

1. `climate`
2. `midea`

Если задача — строить отдельный самодельный термостат, тогда уже нужен `thermostat`.
